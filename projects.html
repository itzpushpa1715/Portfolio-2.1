Single Acting Cylinder

A single-acting cylinder is a type of hydraulic or pneumatic actuator that uses pressure to move in one direction while relying on an external force, such as a spring or gravity, for the return stroke. These cylinders are widely used in applications requiring linear motion, including clamping, lifting, and pressing tasks. Their simple design ensures efficiency, reliability, and ease of maintenance.

Motor control Logic Program

Motor can by started with the push button S1 and stopped with the push button S2. The motor will run when PLC output K1 is active. The motor must stop automatically after the induction sensor S3 has given 100 pulses to prevent motor overheating. If the motor is stopped manually before the pulses reach 100, the pulse counter is reset. The motor must be ready for a restart with the S1 button after a manual or an automatic stop. Note that both buttons are push buttons, which means that the signal state will only be 1 when the button is held. Design the program so that the buttons must only be pressed once for the motor to run and to stop. Some basic blocks that you will need are: SR or RS flip-flops, counter, comparators. Test the program with PLC simulation.

Heating Control program

The resistor is controlled by the output K1. When the temperature measurement TT101 drops below 50°C, the output is activated. When the temperature is over 55°C, the output is turned off. The system also has an alarm signal light, which is connected to the output H1. The alarm light must be on in the following situations: The heating resistor has been on for 5 minutes and the temperature is still under 55°C The heating resistor is on, but the temperature is over 56°C The alarm can be reset with the pushbutton S3, otherwise the signal light must stay on, even if the situation that caused the alarm has been solved. A reset is always required. The temperature measurement TT101 measures the temperature between 0-100°C and since it’s connected to a Siemens analog input, the value that the PLC receives is 0-27648.

PLC and RELAY CONNECTIONS

This PLC program controls three lamps via a relay using a push button. When the push button is pressed, the lamps illuminate. An alarm activates if the relay is not energized while the button is pressed, or if the relay is energized when the button is not pressed. This alarm does not require an acknowledgment signal. The PLC used is a Siemens S7 1214C."
